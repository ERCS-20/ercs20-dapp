# Authentication

This dApp is **wallet-first** (`wallet` skill). Use this for future CeFi admin/login screens.

Minimal fintech login: vertical form, clear errors, 2FA step, session expired modal.

Hybrid products: separate **Connect wallet** vs **Sign in** — do not merge into one ambiguous button.

See `forms`, `security.mdc`.
