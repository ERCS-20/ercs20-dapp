# Dark Mode

Never simple color invert. Use elevation: background → card → popover; borders + `muted-foreground` tiers.

Orbix: dark primary is **pink**, not teal (`globals.css` `.dark`).

Trading: muted bid/ask; low-contrast chart grid.

Test every new component in both themes (`next-themes`).

See `colors` skill.
